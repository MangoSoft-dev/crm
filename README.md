# MangoSoft CRM

## Project Overview
This is the main repository for the MangoSoft CRM system. The project is divided into two main applications:
- **Frontend**: A React-based Single Page Application (SPA) client.
- **Backend**: The API server and core business logic.

## Repository Structure
- [`/frontend`](./frontend/) - Source code for the user interface.
- [`/backend`](./backend/) - Source code for the API and server logic.

## Getting Started
To set up the entire development environment, please follow the specific instructions in each directory:
1. Initialize the database (see `/backend/README.md`).
2. Start the Backend server (see `/backend/README.md`).
3. Start the Frontend application (see `/frontend/README.md`).
